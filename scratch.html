Astronomous

Description

There is a man that flew to mars and crashed there. One day he was able to fix his spaceship and flew back home.
While flying back he finds that aliens have taken over earth. In order to go back, he must dodge the bullets that is being
shot at him. To move, uses the arrow keys. There are three kinds of bullets. The pink bullet, shoots in a linear path way.
The blue bullet shoots in a linear path way too but in triplets. The green bullet shoots at wherever the spaceship is at.

Reflection Questions and Answers

  1. Describe some programming concepts or structures you learned in this unit that you applied in your game.
      While programming I learned more about the "or" and "and" tags. They have become very useful to my game
      because it helped with simplifying the code and getting the sprites to work.
  2. Did you encounter any problems or struggles while working on this project? If you did, how did you solve or overcome them?
      The problem I encounter was making the bullets move to where I needed it to go and how fast it should go. I fixed it by adding a certain
      code to make it go where the ship was. Also, it was a trial and effct to make the bullets not go too fast nor too slow.
  3. What aspects of your game do you like best?
      My favorite aspect of the game is the design and the difficulty of it.
  4. If more time was given for you to work on your game, what would you improve or modify about your game?
      If I had more time I would like to add difficulty levels and fix the scoring system.
